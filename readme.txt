A simple wp-admin UI to browse registered Abilities and disable/enable them.

== Description ==
Lists all abilities registered on the site (by any plugin or core) and lets admins disable or re-enable them.
Disabled abilities are blocked by overriding their permission_callback during registration.

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/ OR install the ZIP via Plugins → Add New → Upload.
2. Activate.
3. Go to Tools → Abilities Inspector.
